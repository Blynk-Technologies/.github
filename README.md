# Blynk-Technologies / .github

This repository contains organization-wide default community health files and shared GitHub Actions workflows for [Blynk Technologies](https://blynk.io).

## What is this repository?

GitHub uses the `.github` repository as a special location to store default files that apply to all repositories in an organization. If a repository doesn't have its own version of these files, GitHub falls back to the ones defined here.

## Contents

- **Community health files** – Default templates for issues, pull requests, code of conduct, and contributing guidelines that apply across all Blynk-Technologies repositories.
- **Shared workflows** – Reusable GitHub Actions workflow definitions used across the organization's repositories.

## About Blynk

[Blynk](https://blynk.io) is an IoT platform that enables developers and businesses to build connected products. Our open-source libraries and tools support a wide range of hardware platforms, including Arduino, ESP32, ESP8266, Raspberry Pi, Zephyr RTOS, MicroPython, and many more.
